Introducing Edgium

Install the latest Linux packages with 1 line.

What is Edgium?

Edgium is a small devops package that syncs your Linux box with the official or most popular community repositories for your favorite apps.

Why?

Most of the time the packages available in our Distros are hopelesly outdated. The only way you can install the latest version is to dig deep into wikis/docs and than execute a lot of similar commands. It gets really painful when you have to repeat the process on different servers over a big period of time (You install PostgreSQL, your app grows and you deploy the next instance in 2 months). Until now the only way to solve this was to create/fork your cookbooks/manifest with popular devops tool like Chef, Puppet, Ansible, etc. There are great tools in their own right and knowing how to use them is valuable knowledge, but sometimes you just want to install the latest PostgreSQL version and play with JSON storage.
