Introducing Edgium

Install up-to-date Linux packages with 1 line.

Why Edgium?

The packages available in our Distros are hopelesly outdated. To install the latest version we have to dig deep into Wikis, Docs, Stack Overflow or some random article/tutorial named "How to install the latest X on [Distro of Choice]". It is really time consuming and painful process.

While reading yet another Stack Overflow thread, I stumbled upon the following statement:

If you want to live on the bleeding edge, then you don't get to choose the easy path. Download the source and build it.

It left a lasting impression on me and I decided to do something about it. This is how Edgium was born.

What is Edgium?

Edgium is a small DevOps tool built around Ansible and everything it does is sync the most popular and regularly updated repository with your yum/repos.d or /etc/apt/sources.list.d.

Edgium is heavily inspired by homebrew for OSX. The major difference is that EDGIUM IS NOT A PACKAGE MANAGER - it uses the underlying package managers(apt, yum).


The example below should help you understand Edgium better:

Before

  1. Want to install recent MongoDB version: apt-get install mongodb
    On Ubuntu 12.04 LTS you will get Mongo 2.0.4, released on 20.03.2012.
  2. Google 'How to install MongoDB 2.6.5 on Ubuntu 12.04', you are lucky - MongoDB is one of the few projects that actually have really good docs.
  3. Go to http://docs.mongodb.org/manual/tutorial/install-mongodb-on-ubuntu/
  4. Follow the steps:
  5. sudo apt-key adv --keyserver hkp://keyserver.ubuntu.com:80 --recv 7F0CEB10
  6. echo 'deb http://downloads-distro.mongodb.org/repo/ubuntu-upstart dist 10gen' | sudo tee /etc/apt/sources.list.d/mongodb.list
  7. sudo apt-get update
  8. sudo apt-get install -y mongodb-org
  9. Awesome. We did it, now let's install nginx and PostgreSQL and repeat the process on 2 more servers.

After

  1. edgium install mongodb
  2. edgium install nginx
  3. edgium install postgresql

Features

  • Installs software not packaged by your distribution
  • Installs up-to-date versions
  • Does not reinvent the wheel - uses official repositories, the most popular community repositories and the tools already available in your distro

Up and running in seconds

Quick-start Instructions

~ $ curl http://edgium.amon.cx/install.sh | bash

~ $ edgium install nginx

# => Will install nginx 1.6.2

&

~ $ apt-get/yum install nginx

# => The next time you will get the latest version

If installing things with curl | bash makes you mad as hell, you can install Edgium the manual way.
